## 👋 About This Project

This is my **personal developer portfolio** — a fully animated, responsive single-page application showcasing my projects, skills, and experience. Built from scratch using the latest React 19 and Vite 8, with buttery-smooth animations powered by both **Framer Motion** and **Anime.js**.

The goal: make a lasting first impression on anyone who visits — whether that's a recruiter, a collaborator, or a curious developer.

<br/>

## ✨ Highlights

- ⚡ **Blazing fast** — Vite 8 with HMR for near-instant dev feedback
- 🎨 **Fully animated** — Page transitions, scroll reveals, and micro-interactions via Framer Motion & Anime.js
- 🖋️ **Typing effects** — Dynamic role display with `react-type-animation`
- 📱 **Fully responsive** — Mobile-first design with Tailwind CSS v4
- 🧭 **Smooth navigation** — Scroll-based routing with `react-scroll`
- 📩 **Contact form** — Integrated with SweetAlert2 for polished feedback dialogs
- 🌙 **Modern aesthetics** — Styled Components + Tailwind for flexible, scoped styling

## 🚀 Getting Started

### Prerequisites

```
Node.js >= 18.x
npm >= 9.x
```

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Irmiya07/My_portfolio_frontend.git
cd My_portfolio_frontend

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Available Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start local dev server with HMR |
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint checks |

<br/>

## 📦 Deployment

This project builds to a standard `dist/` folder and can be deployed to any static hosting service:

```bash
npm run build
# Deploy the /dist folder to your hosting provider
```

**Recommended platforms:** Vercel · Netlify · GitHub Pages · Cloudflare Pages

For Vercel (recommended):

```bash
npm i -g vercel
vercel --prod
```
